# Projeto QA - Sistema de Matrícula Online

## Objetivo
Validar o processo de inscrição e matrícula de candidatos em uma plataforma educacional.

## Escopo
- Testes funcionais
- Validação de regras de negócio
- Testes de API
- Validação em banco de dados

## Artefatos
- Casos de teste
- Registro de bugs
- Consultas SQL
- Coleção Postman

## Competências Demonstradas
- Análise de Requisitos
- Histórias de Usuário
- Critérios de Aceitação
- QA Funcional
- APIs REST
- SQL
- Scrum
